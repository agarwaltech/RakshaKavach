Raksha-Kavach (Healthcare) Android App Development using GenAI
Domain: Healthcare & Industrial Safety
1. Problem Statement
Construction and factory workers often neglect Personal Protective Equipment (PPE) because they find it
uncomfortable or believe they are safe without it. This increases the risk of injuries and accidents at workplaces.
2. Project Objective
The objective of Raksha-Kavach is to create a smart Android application that educates workers about workplace
safety by visually displaying the required safety gear for different tasks and encouraging safe work practices.
3. Proposed Solution
The application will allow workers or supervisors to select a specific task such as welding, trench digging, or height
work. Based on the task selected, the app will display mandatory PPE using animations and icons. It will also
include safety reminders, quizzes, incident reporting, and a safety score system.
4. Main Features
• Task Selector for different industrial activities.
• Safety Gear Checklist (Helmet, Gloves, Boots, Goggles, etc.).
• Risk Meter showing possible injuries if safety gear is ignored.
• Daily Safety Quiz for awareness.
• Incident Log using Room Database.
• Start-of-Day Safety Reminder Notifications.
• Safety Score tracking system.
5. Technologies Used
• Android Studio
• Kotlin 
6. Expected Outcome
The application will help reduce workplace accidents, improve awareness about safety equipment, and encourage
a safety-first mindset among workers in factories and construction sites.
7. Impact
• Improves occupational safety.
• Reduces workplace injuries and accidents.
• Promotes worker dignity and awareness.
• Supports industrial safety culture in India.
8. Conclusion
Raksha-Kavach is a socially impactful Android application that combines technology and safety awareness to
protect workers and promote responsible workplace behavior
